二叉树

特殊二叉树

斜树

所有结点都只有左子树的二叉树叫左子树；所有结点都只有右子树的二叉树叫右子树

满二叉树

在二叉树中，所有结点都存在左子树和右子树，并且所有的叶子结点均在同一层

既有同样深度的二叉树，满二叉树的结点数最多，叶子结点数最多

完全二叉树

对一棵具有n个结点的二叉树按层序编号，如果比编号为i的结点与同样深度的满二叉树中编号为i的结点在二叉树中的位置完全相同，则这棵二叉树称为完全二叉树

叶子结点仅在最下面的两层，具有同样结点数的二叉树，完全二叉树的深度最小

二叉树的性质

1.在二叉树的第i层至多(满二叉树)有2^(i - 1)个结点

2.深度为k的二叉树至多(满二叉树)有2^k - 1个结点

3.对任意一棵二叉树，如果其叶子结点数为n0，度数为2的结点数为n2，则n0 = n2 + 1

证明：

设n1的结点数为n1，则总结点数n = n0 + n1 + n2，以分支数作为桥梁连接

n - 1 = n1 + n2 * 2，可得 n0 = n2 + 1

4.具有n个结点的完全二叉树的深度为⌊log2n⌋ + 1

证明：

由于完全二叉树的叶子结点仅分布在最下面两层，所以2^(k - 1) - 1 < n <= 2^k - 1，可得

2^(k - 1) <= n < 2^k ，k-1 <= log2n < k，所以k = ⌊log2n⌋ + 1

5.按层序编号后，对于结点i，若存在双亲及叶子结点，则双亲的编号为⌊i / 2⌋，左孩子为2*i，右孩子

为2*i + 1
