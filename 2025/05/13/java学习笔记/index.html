Java应用

  • java SE：

java语言的标准版，用于桌面应用的开发

  • java ME:

java语言的小型版，用于嵌入式电子设备或小型移动设备

  • java EE:

java语言的企业版，用于Web方向的开发。

编译运行方式

  • 编译型：C/C++

通过编译器整体编译为二进制机器指令后运行。

  • 解释型：python

通过解释器按行翻译后运行。

  • 混合型：java

先通过编译器整体编译为二进制字节码，然后交给执行器按行在虚拟机上执行。（跨平台的原理）

JDK与JRE

  • JDK：

由JVM（java运行的虚拟机），库函数，开发工具（编译javac、运行java、调试jdb、内存分析jhat等）组成。

  • JRE：

由JVM（java运行的虚拟机），库函数，运行工具组成。（短小）

  • 关系：

JDK包含JRE，JRE包含JVM
